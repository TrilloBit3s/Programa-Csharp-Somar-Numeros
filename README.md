# Programa-Csharp-Somar-Numeros

Programa simples em C# para somar dois números, pode-se usar outros calculos também, mas o básico já mostra todo seu funcionamento.
Caixas de texto para o input das variaveis, um botão de somar os valores e um botão para limpar a caixa de resultados.
